## Quick Start

Just run the `danmin` CLI like this:
```
npx danmin
```
```
Options:
  -V, --version  output the version number
  -c, --career   display career
  -g, --github   open github
  -b, --blog     open blog
  -r, --resume   open resume
  -h, --help     display help for command
```
